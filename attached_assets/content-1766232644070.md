A production-grade cryptocurrency algorithmic trading and backtesting platform built with enterprise-level quality standards.

## Features

[Permalink: Features](https://github.com/canstralian/Crypto-Algo-Trading-Strategy-Optimization-via-Backtesting#features)

- **Multiple Trading Strategies**: MA Crossover, RSI, MACD, Bollinger Bands, and multi-strategy combinations
- **Advanced Backtesting Engine**: High-performance backtesting with realistic market simulation
- **Risk Management**: Position sizing, stop-loss, take-profit, and portfolio optimization
- **Real-time Data**: Integration with multiple cryptocurrency exchanges via CCXT
- **Performance Analytics**: Comprehensive metrics including Sharpe ratio, drawdown, win rate, etc.
- **REST API**: FastAPI-based API for programmatic access
- **Monitoring & Logging**: Structured logging and performance monitoring
- **Security**: Input validation, rate limiting, and secure configuration management
- **Docker Support**: Containerized deployment with docker-compose
- **90%+ Test Coverage**: Comprehensive unit and integration tests

## Quick Start

[Permalink: Quick Start](https://github.com/canstralian/Crypto-Algo-Trading-Strategy-Optimization-via-Backtesting#quick-start)

### Prerequisites

[Permalink: Prerequisites](https://github.com/canstralian/Crypto-Algo-Trading-Strategy-Optimization-via-Backtesting#prerequisites)

- Python 3.9 or higher
- pip or poetry for dependency management
- Docker (optional, for containerized deployment)

### Installation

[Permalink: Installation](https://github.com/canstralian/Crypto-Algo-Trading-Strategy-Optimization-via-Backtesting#installation)

```
# Clone the repository
git clone https://github.com/yourusername/Crypto-Algo-Trading-Strategy-Optimization-via-Backtesting.git
cd Crypto-Algo-Trading-Strategy-Optimization-via-Backtesting

# Create virtual environment
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Copy configuration
cp config/config.example.yaml config/config.yaml
cp .env.example .env

# Edit configuration as needed
vim config/config.yaml
```

### Running a Backtest

[Permalink: Running a Backtest](https://github.com/canstralian/Crypto-Algo-Trading-Strategy-Optimization-via-Backtesting#running-a-backtest)

```
# Run a simple backtest
python scripts/run_backtest.py --strategy ma_crossover --symbol BTC/USDT --timeframe 1h --start 2023-01-01 --end 2023-12-31

# Run with custom configuration
python scripts/run_backtest.py --config config/my_backtest.yaml
```

### Starting the API Server

[Permalink: Starting the API Server](https://github.com/canstralian/Crypto-Algo-Trading-Strategy-Optimization-via-Backtesting#starting-the-api-server)

```
# Development mode
uvicorn src.api.main:app --reload --host 0.0.0.0 --port 8000

# Production mode
gunicorn src.api.main:app -w 4 -k uvicorn.workers.UvicornWorker --bind 0.0.0.0:8000
```

[Permalink: Running Tests](https://github.com/canstralian/Crypto-Algo-Trading-Strategy-Optimization-via-Backtesting#running-tests)

## Available Strategies

[Permalink: Available Strategies](https://github.com/canstralian/Crypto-Algo-Trading-Strategy-Optimization-via-Backtesting#available-strategies)

### 1\. Moving Average Crossover

[Permalink: 1. Moving Average Crossover](https://github.com/canstralian/Crypto-Algo-Trading-Strategy-Optimization-via-Backtesting#1-moving-average-crossover)

Generates signals when short-term MA crosses long-term MA.

### 2\. RSI (Relative Strength Index)

[Permalink: 2. RSI (Relative Strength Index)](https://github.com/canstralian/Crypto-Algo-Trading-Strategy-Optimization-via-Backtesting#2-rsi-relative-strength-index)

Identifies overbought/oversold conditions.

### 3\. MACD (Moving Average Convergence Divergence)

[Permalink: 3. MACD (Moving Average Convergence Divergence)](https://github.com/canstralian/Crypto-Algo-Trading-Strategy-Optimization-via-Backtesting#3-macd-moving-average-convergence-divergence)

Signal line crossovers and histogram analysis.

### 4\. Bollinger Bands

[Permalink: 4. Bollinger Bands](https://github.com/canstralian/Crypto-Algo-Trading-Strategy-Optimization-via-Backtesting#4-bollinger-bands)

Price breakouts and mean reversion.

### 5\. Multi-Strategy

[Permalink: 5. Multi-Strategy](https://github.com/canstralian/Crypto-Algo-Trading-Strategy-Optimization-via-Backtesting#5-multi-strategy)

Combines multiple strategies with configurable weights.

## Configuration

[Permalink: Configuration](https://github.com/canstralian/Crypto-Algo-Trading-Strategy-Optimization-via-Backtesting#configuration)

Configuration is managed through YAML files and environment variables:

```
# config/config.yaml
exchange:
  name: binance
  sandbox: true

backtesting:
  initial_capital: 10000
  commission: 0.001
  slippage: 0.0005

risk_management:
  max_position_size: 0.1
  stop_loss: 0.02
  take_profit: 0.05
```

## Performance Metrics

[Permalink: Performance Metrics](https://github.com/canstralian/Crypto-Algo-Trading-Strategy-Optimization-via-Backtesting#performance-metrics)
